# Line-by-Line Explanation of Frontend `main.js`

This file serves as the main JavaScript controller for the desktop app's client-side user interface. It binds GUI elements to Go backend functions, handles view transitions, sanitizes text inputs, parses message packets into interactive widgets, and updates download/upload states.

#### Lines 110–139: Send Message Event Handler

```javascript
const triggerSendMessage = async () => {
    const text = chatInput.value.trim();
    if (!text) return;

    chatInput.value = '';

    if (text.startsWith('/get ')) {
        const file = text.substring(5).trim();
        if (file) {
            try {
                await DownloadFile(file);
                appendSystemMessage(`Requested download of "${file}"...`);
            } catch (err) {
                appendSystemMessage(`Download error: ${err.message}`);
            }
        }
        return;
    }

    try {
        await SendMessage(text);

    } catch (err) {

    }

```
* **Slash Commands / Downloads**: If the input starts with `/get `, it requests a download directly rather than posting a text bubble.
* **Message Dispatch**: Invokes Go `SendMessage` to write to the TCP socket.
* **Self-Posting (No-Echo)**: Because the server does not bounce outgoing messages back to the sender, the JS controller manually appends the message as a right-aligned (`outgoing`) bubble immediately on success.

#### Lines 179–232: Chat Message UI Builder & Shared File Card Parser
```javascript
function appendChatMessage(sender, text, isOutgoing) {
    const fileUploadRegex = /^(.*) uploaded a file: (.*) \((\d+) bytes\)\. Type '\/get (.*)' to download it\.$/;
    const fileMatch = text.match(fileUploadRegex);

    if (fileMatch) {
        const uploader = fileMatch[1];
        const filename = fileMatch[2];
        const bytes = parseInt(fileMatch[3], 10);
        appendFileCard(uploader, filename, bytes);
        return;
    }

    const row = document.createElement('div');
    row.className = `message-row ${isOutgoing ? 'outgoing' : 'incoming'}`;
    row.innerHTML = `
        <span class="message-sender">${isOutgoing ? 'You' : sender}</span>
        <div class="message-bubble">${escapeHTML(text)}</div>
    `;

    messagesWindow.appendChild(row);
    scrollToBottom();
}
```
* **Interactive File Upgrades**: Uses a regular expression check to match system-generated file uploads messages from the TCP server. Instead of displaying the raw terminal text command, it renders a custom File Card card containing document icons, sizes, and a **Download** action button.
